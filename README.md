# Сайт КАЛЕНДАРЬ

Календарь ивентов: стримы, ХЕРНЯТурики, съёмки, спонсорские турниры.
Статический сайт на HTML + CSS + JavaScript, без сборки и зависимостей.
Живёт на GitHub Pages → [klaxix.xyz](https://klaxix.xyz)

## Как обновить ивенты

1. Открой `/editor/` на сайте.
2. Добавь/измени ивенты (там же вставляется ссылка на VOD), нажми **Скачать events.json**.
3. Залей `events.json` в корень репозитория вместо старого.

На GitHub это можно сделать прямо в браузере: открой репозиторий → зайди в
`events.json` → карандаш (Edit) → вставь новое содержимое → Commit.
Сайт обновится через 1–2 минуты.
