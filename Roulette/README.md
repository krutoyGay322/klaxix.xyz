# Рулетка — DBD casino roulette

Автономная слот-машина по Dead by Daylight: перки и персонажи.
Работает без сервера — просто открой `index.html` в браузере
(подходит и как browser source в OBS).

**Крутить — рывком за красный шар** (он пульсирует, когда готов).
Панель только выбирает режим и громкость:

- **Перки: выживший / убийца** — каждый рывок роллит 4 случайных уникальных перка.
- **Рулетка: выжившие / убийцы** — каждый рывок один спин: промах — красная
  вспышка, джекпот — все 4 барабана на победителе с золотым свечением.
  Крути, пока не выпадет.

Панель прячется по **✕** (вернуть — кнопка 🎰 в углу), громкость и мут
действуют на все звуки сразу и переживают перезагрузку.

Устройство, пересборка и обновление данных — в [docs/roulette.md](../docs/roulette.md).
