# Рандомайзер Команды

Рандомизирует перки и предметы для убийцы и четырёх выживших. Ассеты (перки, портреты, предметы) берутся из `../CaptainShop`, русские названия — из словарей `../Roulette/data`.

## Файлы

- `index.html` / `style.css` / `script.js` — сама страница (ванильный JS, без сборки).
- `data.json` — сгенерированный список перков/персонажей/предметов.
- `tools/build-data.mjs` — генератор `data.json`. После добавления новых картинок в CaptainShop запусти из корня репозитория:

  ```
  node TeamRandomiser/tools/build-data.mjs
  ```

  Скрипт напишет, для каких файлов не нашлось русского названия (добавь их в словари Roulette или в overrides внутри скрипта).

- `_legacy/` — исходный прототип из DevComponents.

## Механика

- Тир перка = папка Tier 1/2/3 в магазине капитана (I обычный / II редкий / III легендарный). Шансы: 15% на III, 30% на II, остальное I.
- Перки не повторяются: у убийцы — в рамках его четырёх слотов, у выживших — во всей команде (поэтому перки можно перетаскивать между слотами и игроками).
- Предметы уникальны между выжившими; редкость предмета как в DBD — коричневый/жёлтый/зелёный/фиолетовый (шансы 35/30/25/10). Предметы тоже можно перетаскивать между выжившими.
- Убийцы и выжившие в пикерах отсортированы по дате выхода в DBD (порядок задан в `tools/build-data.mjs`).
- Выбор персонажей и результаты сохраняются в localStorage.
